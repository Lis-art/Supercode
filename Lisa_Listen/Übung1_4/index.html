ul vs. ol

  • unordered list
    • HTML-tag:ul
    • listet Punkte mit bullet point auf
    • Beispiele
      • To-Do Liste
      • ...
      • ...
    • Weiterführende Links
  • ordered List
    • HTML-tag:ol
    • listet Punkte mit Nummerierungen auf
    • Beispiele
      1. Inhaltverzeichnis
      2. ...
      3. ...
    • Weiterführende Links
